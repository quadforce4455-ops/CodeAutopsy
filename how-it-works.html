The Autopsy Process

Everything CodeAutopsy can do, step by step.

1

Sign In

Create an account with email and password or jump straight in with GitHub OAuth. Your review history and session are tied to your account.

Firebase Auth GitHub OAuth
2

Paste or Import Code

Paste code directly into the editor, drag and drop a file, or import straight from a GitHub repository. Language is auto-detected on paste. Supports 8+ languages.

C / C++ Python JavaScript TypeScript Java Go Rust
3

Choose Your Mode

Pick the analysis mode that fits your need:

CODE AUTOPSY

Bug detection, security vulnerabilities, time & space complexity, improved code

CODE EXPLAINER

Line-by-line breakdown of what your code does + improvement suggestions

CODE SMELL

Detects bad patterns & anti-patterns — long methods, magic numbers, deep nesting, duplication & more

PR REVIEW

Paste a GitHub PR URL — fetches the diff automatically and reviews every changed file for bugs, security issues & suggestions

4

Run the Analysis

Gemini AI performs a deep static analysis — scanning for critical bugs, security holes, code smells, and complexity problems. Usually done in under 10 seconds.

Gemini Flash Static Analysis Security Scan
5

Get Your Diagnosis

See a full breakdown of your code's health — output depends on the mode you chose:

  • Autopsy — quality score, bug cards, security issues, complexity, improved code + diff view
  • Explainer — summary, line-by-line breakdown, improvement suggestions
  • Smell — health score, smell cards by category, metrics, refactored code
  • PR Review — verdict, per-file issue cards, overall score, summary
  • Click any issue card to jump to that line in the editor
  • Every result saved automatically to History
6

Review History & Export

Every analysis is saved automatically. Visit the History page to see your score trends over time, filter by language or mode, re-run old reviews, and export full reports as markdown.

Score Trends Filter & Sort Export Report Share Link
